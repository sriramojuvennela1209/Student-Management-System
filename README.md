# Student Management System

## Overview

A Java-based Student Management System using JDBC and MySQL.

## Features

* Add Student
* View Students
* Update Student
* Delete Student
* Search Student

## Technologies Used

* Java
* JDBC
* MySQL

## Database

Database Name: student_db

Table: students

Columns:

* id
* name
* age
* course
